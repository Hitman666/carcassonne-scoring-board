  • How did I do it?
Since I'm a big fan of Carcassonne board game and I happen to came accross this unique project on the web, I just had to make one for myself :). Electronic counter proved not only to be great for tracking Carcassonne scores, but we also used it to track scores in Qwirkle.
The idea for a web application came naturally since the electronic counter was too cumbersome to carry around.
Now, since this was done in a really short period of time, not a lot effort has been given to tuning it up, that's why I encourage you to give suggestions or even help out on GitHub.
You can read about the dev process on my blog. In short, I was using Yeoman with Angular generator and it's my first venture down the Angular lane. You can fork the code on GitHub and help me add new features and make it look better on more viewports.
