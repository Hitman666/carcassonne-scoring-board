  • How did I do it?
Since I'm a big fan of Carcassonne board game and I happen to came accross this unique project on the web, I just had to make one for myself :). This was 6 years ago (2007), and since then this electronic counter prooved not only great for Carcassonne scoring, but we also use it to track score in Qwirkle.
The idea for a web app came naturally since the electronic counter was too cumbersome to carry around. So, why not make an app which would easily work on any smartphone? Boom - a day later the basic functional app was there.
Now, since this was done in a really short period of time, not a lot effort has been given to tuning it up, that's why I encourage you to give suggestions or even help out on github.
You can read about the dev process on my blog. In short, I was using Yeoman with Angular generator and it's my first ventuare down the Angular lane. You can fork the code on Github and help me add new features and make it look better on more viewports.
